# Библиотека для решения систем линейных уравнений (СЛАУ)

**Header-only библиотека на C++ для решения систем линейных уравнений вида Ax = b.**

>[!note]
>Проект ориентирован на численные методы, работу с разреженными матрицами и сравнение алгоритмов по сходимости и производительности.

---

## Реализованный функционал

### 1. Базовые операции с векторами

Перегружены основные операторы для `std::vector<T>`:

- Сложение и вычитание векторов
- Умножение вектора на число
- Скалярное произведение
- Умножение матрицы на вектор

---

### 2. Типы матриц

#### Плотная матрица (DenseMatrix)

- Хранение в виде двумерного контейнера
- Доступ к элементам через оператор `(i, j)`
- Умножение на вектор

#### Разреженная матрица в формате CSR

- Реализация формата Compressed Sparse Row
- Конструирование из DOK-представления
- Быстрое умножение на вектор
- Поддержка транспонированного умножения (необходимо для BiCG)
- Эффективное хранение памяти для разреженных задач

![Картинку съел Ульяновский автомеханический завод](https://github.com/GlebLarkin/Systems-of-linear-equations/blob/main/data/t(n).png) 
Как видно из графиков, ассимтотики умножения обычной и CSR матрицы на вектор одинаковые, но CSR матрица умножает в среднем быстрее. Но если ее плоность равна 1.0, то скорость умножения такая же, как и для плотной матрицы, что в общем (и целом) ожидаемо.

#### Трёхдиагональная матрица

- Хранение трёх диагоналей
- Доступ к элементам
- Используется в алгоритме прогонки

---

### 3. Прямые методы

- Алгоритм прогонки (метод Томаса) для трёхдиагональных матриц
- QR-разложение, LU-разложение для CSR-матрицы

---

### 4. Классические итерационные методы

- Метод простых итераций
- Метод простых итераций с ускорением Чебышёва
>[!note]
>К сожалению, для работы метода необходимо знать максимальное и минимальное собыственные значения матрицы, задающей СЛАУ. Максимальное собственне значение можно оценить с помощью функции Estimate_max_eigenvalue, но минимальное нужно передавать самостоятельно.
- Метод Гаусса–Зейделя
- Метод Якоби
- Метод верхней релаксации (SOR)
